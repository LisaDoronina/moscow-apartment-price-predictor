# Moscow Apartment Price Predictor

A Streamlit web application that predicts apartment prices in Moscow using machine learning.

## Features

- Predict apartment prices based on various features
- User-friendly web interface
- Machine learning model trained on real estate data

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/moscow-apartment-predictor.git
```

## Running 
To launch the application, write in the terminal:
```bash
streamlit run app/app.py
```
